# 🗓️ Syntax Schedule

A modern, printable **HTML/CSS class schedule** optimized for both screen and paper.  
It uses a **CSS grid layout**, and **print-optimized styles** to produce a clean and professional timetable.

---

## ✨ Features

- 📅 **Fully customizable** weekly class schedule  
- 🎨 **Gradient course colors** for better visual distinction  
- 🖨️ **Print-optimized** with perfect A4 alignment  
- 📐 **Responsive CSS grid** layout  
- 🧭 **Clear time indicators** and room information  
- 🖌️ Simple to edit and adapt for any semester or program
